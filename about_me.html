Web developer:

A web developer is a programmer or a coder who specilize in,or is sepcifially
engaged in, the developement of World Wide Web applications using a client-server model. The
applications typically use HTML, CSS and Java Script in hte client, PHP, ASP.NET, Python, Node.Js
or Java in the server, and http for communitcations between client and server. A web developer may
specilize in cilent-side applications (front-end developer) server-side applications
(back-end developer) or both (full-stack developer).

A web content management system may be used to develop and maintain web applications.

Web design and developer is an umbrealla term that describes the process of creating a website.
Like the name suggest, it involves two major skill sets: Web design and web development. Web
design determines the look and feel of a website, while web development determines how it functions.
Because there isn't always a hard line that seprates the two roles, the titles are often used
interchageably. As the web continues to evolve, so do the roles.
In the almost 30 years since the first website was created, numerous job titles have emerged to
describe the various skill sets used to create a webisite, with more coming out every year. These titles
often overlap, and their meanings change from company. It's enough to make your head spin.
